🧊 STL 파일 직접 업로드

STL 파일을 그대로 올리면 사이트가 자동으로 슬라이싱합니다. 별도 프로그램 없이 바로 신청 가능.

  • ✔ 프로그램 설치 불필요, 가장 빠름
  • ✔ 받침대 없어도 되는 간단한 모델에 적합
  • ✘ 서포트(지지대) 자동 추가 안 됨
업로드 페이지로 →
방법 2 · 정밀

⚙️ Bambu Studio에서 직접 슬라이싱

Bambu Studio에서 직접 슬라이싱 후 .3mf 파일을 업로드합니다. 아래 가이드를 따라하세요.

  • ✔ 서포트 직접 설정 가능
  • ✔ 크기·방향·속도 세밀하게 조절
  • ✘ Bambu Studio 설치 필요

아래는 방법 2 (Bambu Studio 슬라이싱) 상세 가이드입니다.

STL 모델 → Bambu Studio에서 슬라이싱 → .3mf 파일 → 사이트에 업로드

1

Bambu Studio 설치하기

우리 학교 프린터는 Bambu Lab A1입니다. 전용 프로그램인 Bambu Studio를 쓰면 프린터 설정이 자동으로 맞춰져서 가장 쉽습니다.

  1. 공식 다운로드 페이지로 이동: bambulab.com/en/download/studio
  2. 본인 컴퓨터 운영체제에 맞는 버전 다운로드 (Windows / macOS)
  3. 다운로드한 설치 파일을 실행하고 안내에 따라 설치
  4. 설치 후 Bambu Studio 실행
계정 없어도 됩니다. 처음 실행 시 로그인하라고 나오지만, 슬라이싱과 .3mf 내보내기는 로그인 없이 가능합니다. 건너뛰어도 OK.
2

프린터 선택하기

처음 실행하면 어떤 프린터를 쓸지 물어봅니다. 여기서 Bambu Lab A1을 선택하세요. (A1 mini 아님! 그냥 A1입니다.)

  1. 프린터 목록에서 A1 선택
  2. 노즐은 기본값 0.4mm 그대로
  3. 이미 다른 프린터로 설정돼 있다면, 왼쪽 위 프린터 아이콘을 눌러 A1로 변경
프린터를 잘못 고르면 출력 크기·설정이 안 맞아 실패할 수 있어요. 꼭 A1인지 확인!
3

모델 불러오기

출력할 3D 모델 파일(.stl 또는 .3mf, .obj, .step)을 불러옵니다.

  1. 왼쪽 위 파일 → 가져오기(Import) 또는 모델 파일을 화면에 끌어다 놓기
  2. 화면 가운데 회색 판(베드) 위에 모델이 올라옵니다
  3. 마우스로 회전(우클릭 드래그)·확대(스크롤)하며 모델 확인
모델은 MakerWorld, Thingiverse 등에서 무료로 받거나, 직접 모델링한 STL을 쓰면 됩니다.
4

크기·위치 확인하기

모델이 베드 안에 들어가는지, 너무 작거나 크지 않은지 확인합니다. A1 베드 크기는 256 × 256mm입니다.

  1. 모델 클릭 → 왼쪽 도구에서 크기 조절(Scale)로 크기 변경 가능
  2. 모델이 베드(회색 영역)를 벗어나면 빨갛게 표시됩니다 → 크기를 줄이세요
  3. 이동(Move) 도구로 베드 가운데에 배치
모델이 클수록 출력 시간이 길어집니다. 처음이라면 작게(5~8cm) 시작하길 추천! 큰 모델은 몇 시간씩 걸려요.
5

필라멘트(색상) 선택하기

어떤 색·재질로 출력할지 정합니다. 우리 프린터에는 AMS(자동 색상 교체 장치)가 달려 있어 여러 색이 들어있습니다.

  1. 업로드 페이지나 대시보드에서 현재 프린터에 들어있는 색상을 먼저 확인하세요
  2. 오른쪽 위 필라멘트 설정에서 같은 색·재질(보통 PLA)을 선택
  3. 처음이라면 단색 한 가지로만 하는 걸 추천 (멀티컬러는 복잡해요)
대시보드에 표시된 AMS 색상과 다른 색을 고르면, 실제 출력 색이 달라질 수 있어요. 지금 프린터에 든 색에 맞추세요.
6

슬라이싱 하기

이제 번역 버튼을 누를 차례입니다.

  1. 오른쪽 위 슬라이스(Slice plate) 버튼 클릭
  2. 잠시 계산하고 나면, 화면이 출력 미리보기로 바뀝니다
  3. 예상 출력 시간과 필라멘트 사용량이 표시됩니다
기본 설정(레이어 0.2mm, 인필 15%)이면 대부분 잘 나옵니다. 처음엔 설정을 바꾸지 말고 기본값 그대로 두세요.
7

.3mf 파일로 내보내기

마지막 단계! 슬라이싱 결과를 파일로 저장합니다.

  1. 왼쪽 위 파일(File) 메뉴 클릭
  2. 플레이트의 모든 데이터를 내보내기(Export plate sliced file) 선택
  3. 파일 이름을 본인 이름이나 모델 이름으로 저장 (예: luke_keychain.3mf)
  4. 저장 위치를 기억하세요 — 곧 업로드할 파일입니다
반드시 슬라이싱을 먼저 한 뒤 내보내야 합니다. 슬라이스 안 한 파일은 출력 정보가 없어요. 6번 → 7번 순서 지키기!

완성! 이제 업로드하세요

저장한 .3mf 파일을 업로드 페이지에서 올리고, 프린터를 선택한 뒤 신청하면 됩니다. 관리자 승인 후 차례가 되면 출력됩니다.

홈으로 파일 업로드하러 가기

자주 묻는 문제

모델이 베드보다 커요 / 빨갛게 표시돼요

모델이 A1 출력 범위(256×256mm)를 벗어난 겁니다. 4번 단계의 크기 조절(Scale)로 크기를 줄이세요. 비율은 보통 자동으로 유지됩니다.

출력 시간이 너무 길게 나와요

모델이 크거나 인필이 높으면 시간이 늘어납니다. 모델 크기를 줄이거나, 처음이라면 더 작은 모델로 연습하세요. 몇 시간짜리 출력은 관리자와 상의가 필요할 수 있어요.

서포트(support)가 뭔가요? 켜야 하나요?

공중에 떠 있는 부분을 받쳐주는 임시 구조물입니다. 평평한 바닥에 잘 서는 모델은 필요 없지만, 튀어나온 부분이 많으면 켜는 게 좋습니다. 잘 모르겠으면 기본값(꺼짐)으로 출력해보고, 실패하면 서포트를 켜서 다시 시도하세요.

Bambu Studio로 슬라이싱했는데 어떤 파일을 올려야 하나요?

.3mf 파일을 올려주세요. 파일 메뉴 → 플레이트의 모든 데이터를 내보내기(Export plate sliced file)로 저장한 파일입니다. .3mf에는 색상·플레이트 정보가 모두 담겨 있어 우리 시스템·프린터와 가장 잘 맞습니다.

색을 여러 개 쓰고 싶어요

AMS 멀티컬러는 가능하지만 설정이 복잡하고 실패 확률도 높습니다. 처음에는 단색으로 익숙해진 뒤 도전하는 걸 추천합니다. 멀티컬러가 꼭 필요하면 관리자에게 문의하세요.
